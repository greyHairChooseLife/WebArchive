# Features

기능 범위. MVP / 나중 / 비전으로 구분.

## 비전

북마크 대체를 넘어 **탭 매니저 + 아카이브**. 지금 열린 탭들을 여러 관점으로 관리하다가, 가치 있는 걸 영구 보관(note)으로 승격한다.

-   note = 북마크를 넘어선 "웹페이지 클립 문서": 리치 에디터 본문(텍스트+이미지 인라인, Notion식 블록), 썸네일·페이지 메타(og:image)로 enrich.
-   group(상호배타) + 전역 tag(다대다)로 대량 북마크를 체계적으로 관리.

핵심 차별점:
-   기존 북마크(폴더 트리)의 "한 곳에만 넣을 수 있음" 한계를 전역 다대다 tag로 해소.
-   탭 매니징(라이브)과 아카이빙(저장)을 한 패널에서 연결.

## MVP 범위

### 1. 라이브 탭 멀티 관점 뷰

-   지금 브라우저에 열린 모든 탭을 가져와 표시 (저장 안 함, 라이브).
-   단일 탭 리스트 + 상단 그룹핑 토글로 한 번에 한 관점:
    -   **윈도우별**: `windowId`로 묶기
    -   **도메인별**: hostname으로 묶기
    -   **시간대별**: 마지막 본 지 경과(`lastAccessed`) 기준 5구간 — 최근(5분 이내) / 1시간 이내 / 오늘(1~24시간) / 며칠 전(1~7일) / 오래됨(7일 이상). 빈 구간 숨김.
-   탭 카드: favicon + title + 도메인 + "마지막 본 지" 표시. 체크박스로 다중 선택.
-   "탭으로 점프"(클릭 시 해당 탭 활성화)는 나중.

### 2. 탭 가져오기 → note 저장

-   탭 카드 체크 → "가져오기" → 대상 group 지정(기존 group 선택 또는 새 group 생성) → 선택 탭들이 그 group의 note로 저장.
-   "현재 윈도우 전체 / 모든 윈도우 전체"는 일괄 가져오기 버튼.
-   group은 필수, tag는 사후(라이브러리에서 note별로). 미분류·임시 보관소 없음.
-   URL 중복 검사 없음 — 같은 탭을 또 가져오면 그대로 또 저장됨.

### 3. 계층 관리

-   group CRUD (group 삭제 시 하위 note 통째 삭제, 확인 모달).
-   tag CRUD (전역. tag 삭제 시 라벨만 제거, note 유지).
-   note CRUD (group 소유, 전역 tag 여러 개 부착 가능, tag 없어도 됨).

### 4. note 저장 (개별)

-   수집 필드: url, title, favicon, createdAt, updatedAt (`chrome.tabs`로 수집).
-   memo: plain text 한 줄 (body에 저장).

### 5. 탐색 / 표시 (라이브러리 뷰)

-   group 목록 → group 선택 → 해당 group의 note 표시.
-   note 정렬: 최신 저장순(createdAt 내림차순) 고정.
-   tag 필터: tag 선택 시 기본은 **전역**(모든 group 가로질러 그 tag 달린 note). "현재 group 내에서만" 체크박스로 범위 한정 가능. 다중 tag(AND/OR)는 나중.
-   note는 favicon + title + url 카드 형태(텍스트 위주).

### 6. 데이터 관리

-   JSON export / import (수동 백업·이전).
-   import는 통째 교체(restore): 기존 데이터를 비우고 파일 내용으로 전체 대체 (D14). 부분 병합 없음.

## 나중 (자리는 비워둠)

-   **presence 표시**: 라이브러리에서 지금 열려 있는 note 표시 / 탭 뷰에서 이미 저장된 탭 표시. URL 매칭 토대가 필요해 MVP에서 제외.
-   **탭으로 점프**: 탭 카드/ note 클릭 시 이미 열린 탭으로 이동(`chrome.tabs.update`).
-   **중복 정리**: 비슷한 URL note가 쌓일 때 중복 표시·일괄 삭제.
-   **다중 tag 필터**: AND/OR 조합.
-   **리치 body**: `format: 'blocks'` 기반 Notion식 에디터, 이미지 인라인.
-   **썸네일 / og:image**: content script로 페이지 메타·본문 추출.
-   **페이지 클립**: 본문 발췌·전체 캡처.
-   **기존 북마크 import**: 브라우저 북마크 가져오기, 원본 시각 보존.
-   **수동 시각 입력**: createdAt 사용자 지정.
-   **검색**: 전역 tag·텍스트 기반 검색.
-   **컨텍스트 메뉴 / 단축키**: 우클릭 "담기", 키보드로 패널 열기·빠른 저장 (SW 역할 확장 필요).
-   **가상 스크롤**: note 대량 시 react-window.
-   **IndexedDB 마이그레이션**: 데이터 ~1만 개 한계 도달 시.
-   **자동 동기화**: 클라우드 / 기기 간.
